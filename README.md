# DoStaffkin

Веб-приложение для управления доставкой: создание заказов, расчёт стоимости и отслеживание статуса.

## Функционал

- **Главная страница** — презентация сервиса
- **Создание заказа** — форма с интеграцией Яндекс.Карт для выбора маршрута, выбор размера груза и скорости доставки
- **Отслеживание** — поиск информации о доставке по номеру заявки

## API интеграция

Базовый URL: `https://zhar-anna.github.io/dostaffkin/`

- `POST /create` — создание новой доставки
- `GET /info?id={id}` — получение информации о доставке

## Конфигурация переменных окружения

### Яндекс.Карты API

Проект использует Яндекс.Карты API для функционала создания заказа. API ключи безопасно хранятся в файле `.env`:

#### Установка

1. Скопируйте файл `.env.example` в `.env`:
   ```bash
   copy .env.example .env
   ```

2. Заполните ваши API ключи в файле `.env`:
   ```env
   YANDEX_MAPS_API_KEY=ваш_api_ключ
   YANDEX_MAPS_SUGGEST_API_KEY=ваш_suggest_api_ключ
   ```

3. Получить API ключи можно на: https://developer.tech.yandex.com/services/api/maps/

#### Как это работает

- Переменные из `.env` автоматически загружаются при запуске `npm start` или `npm build`
- Скрипт `scripts/load-env.js` подставляет значения в `src/index.html`
- **Важно:** `.env` файл исключен из системы контроля версий (`.gitignore`), поэтому каждый разработчик должен создать свой собственный `.env` файл
