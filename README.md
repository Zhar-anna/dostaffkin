# DoStaffkin

Веб-приложение для управления доставкой: создание заказов, расчёт стоимости и отслеживание статуса.

## Функционал

- **Главная страница** — презентация сервиса
- **Создание заказа** — форма с интеграцией Яндекс.Карт для выбора маршрута, выбор размера груза и скорости доставки
- **Отслеживание** — поиск информации о доставке по номеру заявки

## API интеграция

Базовый URL: `https://zhar-anna.github.io/dostaffkin/`

- `POST /create` — создание новой доставки
- `GET /info?id={id}` — получение информации о доставке

## Быстрый старт

### Установка

```bash
npm install
```

### Запуск

**Разработка:**
```bash
npm start
```
Приложение откроется на `http://localhost:4200`

**Сборка:**
```bash
npm build
```

**Режим наблюдения:**
```bash
npm watch
```

**Тестирование:**
```bash
npm test
```

## Требования

- Node.js 18+
- npm 10.9.2+
